E-Commerce ELT Data Warehouse (Orchestration & DWH)

Data Engineering | MLOps Prep

The Challenge: Building a production-ready infrastructure to transform raw, fragmented retail data into high-performance business marts while ensuring 100% data recovery and quality.

The Action (7-Stage Medallion Architecture):

  • Orchestration: Engineered an idempotent 7-stage DAG in Apache Airflow to automate ingestion, validation, and transformation.
  • Data as Code (dbt): Developed a centralized transformation layer using dbt, implementing Incremental Materialization to reduce compute load by 90%.
  • Schema Design: Structured a robust Star Schema in PostgreSQL, utilizing MD5 surrogate keys and integer-based date keys for sub-second query performance.
  • Governance: Integrated automated quality gates and dbt-native tests to enforce referential integrity across the pipeline.

The Impact: Delivered a fully containerized (Docker) end-to-end pipeline that provides real-time visibility into Customer Lifetime Value (CLV) and sales trends, ready for immediate BI consumption or downstream ML training.
